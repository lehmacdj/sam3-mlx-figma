## Overview
Uses SAM3 to segment images in Figma.

## Usage
1. Compile the figma plugin code with (complete [autogenerated Figma plugin instructions](./figma-plugin-instructions.md)):
   ```bash
   npm install -g typescript
   npm install --save-dev @figma/plugin-typings
   npm run watch # to compile code.js continuously
   ```
2. Select "Plugins > Development > Import plugin from manifest" selecting manifest.json to import the plugin in Figma.
3. Run sam3-mlx/app/run.sh to startup the python/next.js services used by the plugin on localhost.
4. Select "Plugins > Development > figma-sam3" to run the plugin
5. Use the pop-up UI / select on the image to specify what to extract
